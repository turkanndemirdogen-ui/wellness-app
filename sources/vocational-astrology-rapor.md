# Vocational Astrology — Arketip, Yetenek ve Çağrı Raporu

> **Eser:** *Vocational Astrology* — Judith Hill
> **⚠ Modül notu:** Bu kaynak **medikal astroloji / herbalizm değildir.** Meslek-kariyer astrolojisidir (natal haritadan yetenek, çağrı, arketip). Bu rapor **arketip/kişilik quizi** ve **natal-chart kişiselleştirme algoritması** modüllerini besler — herbalizm kasasına eklenmez.
> **Yöntem:** Taranmış PDF (OCR ile metne çevrildi). Gezegen→meslek listeleri, üç vocational ev ve element/mode iş tarzları eserden doğrudan alınmıştır.

## 1. Eserdeki temel ilkeler

| İlke | Eserin anlatımı |
|---|---|
| **Üç vocational ev** | Kariyeri **10., 6. ve 2. evler** belirler ("artha" üçlüsü). 10. ev en güçlüsü (konum, itibar, çağrı); 6. ev işin kendisi (iş, beceri, iş ortamı); 2. ev kazanç gücü ve gelir kaynağı. |
| **Midheaven (MC)** | Hiyerarşinin tepesinde. Yaşam çağrısını, mesleği, statüyü ve en büyük başarıları yansıtır. MC'ye en yakın gezegen veya MC burcunun yönetici gezegeni kariyeri tarif edebilir. |
| **Üç anahtar nokta** | Kariyer okuması üç katmanı birleştirir: Güneş burcu+evi (iç amaç), Ay burcu+evi (duygusal uygunluk), kariyer yönetici gezegeni. |
| **Gezegen = çağrının doğası** | Her gezegen bir meslek ailesi yönetir; haritadaki baskın gezegen mesleğin temasını verir. |
| **Element + Mode = çalışma tarzı** | Element (ateş/toprak/hava/su) işin alanını; mode (öncü/sabit/değişken) iş davranışını verir. |

## 2. Gezegen → meslek / yetenek arketipi

Eserin Ch4 listeleri (çekirdek). Uygulamanın quiz/arketip motoru için doğrudan veri.

| Gezegen | Evrensel ilke (arketip) | Temsilî meslekler / yetenekler |
|---|---|---|
| **Güneş** (Aslan) | Merkez, otorite, benlik | Liderler, yöneticiler, patronlar, oyuncular, girişimciler, koçlar, bir faaliyetin merkezindeki kişiler |
| **Ay** (Yengeç) | Değişim, alıcılık, besleyicilik, duygu | Aşçılar, hemşireler, çocuk bakıcıları, denizciler, şarkıcılar, fotoğrafçılar, romancılar, ebeler, şifacılar (herbalist), psişikler; halkla ilişki |
| **Merkür** (İkizler/Başak) | Bağlantı, iletişim, ilişkilendirme, veri | Yazarlar, gazeteciler, çevirmenler, muhasebeciler, matematikçiler, programcılar, editörler, öğretmenler, satıcılar, el becerisi gerektiren çalgıcılar |
| **Venüs** (Boğa/Terazi) | Uyum, güzellik, çekim | Sanatçılar, tasarımcılar, kuaförler, mücevherciler, şefler, dekoratörler, danışmanlar, modacılar, çiçekçiler, estetisyenler |
| **Mars** (Koç, Akrep ortak) | Savaş, savunma, enerji, cesaret | Cerrahlar, sporcular, askerler, polisler, mühendisler, mekanikçiler, avukatlar, itfaiyeciler, inşaatçılar, fiziksel-emek meslekleri |
| **Jüpiter** (Yay, Balık ortak) | Genişleme, iyilikseverlik, yüceltme | Politikacılar, yayıncılar, öğretmenler, vaizler, yöneticiler, hayır kurumu başkanları, ithalat-ihracat, pilotlar, finansörler |
| **Satürn** (Oğlak/Kova) | Düzen, biçim kurma, yasa | Mimarlar, çiftçiler, araştırmacılar, bilim insanları, hâkimler, muhasebeciler, mühendisler, zanaatkârlar, küratörler, bürokratlar |
| **Uranüs** (Kova ortak) | Kaos, beklenmedik, aydınlanma, özgürleşme | Astrologlar, mucitler, astronomlar, bilgisayar uzmanları, elektrik işleri, fikir insanları, devrimciler, çizerler |
| **Neptün** (Balık ortak) | Çözülme, biçimin erimesi, duyarlılık | Müzisyenler, şairler, fotoğrafçılar, ressamlar, homeopatlar, enerji şifacıları, psişikler, hipnoterapistler, denizciler, şamanlar |
| **Plüton** (Akrep ortak) | İfşa, arınma, kökten dönüşüm | Araştırmacı-dedektifler, kriminologlar, nükleer/fizik, arkeologlar, genetikçiler, geri dönüşüm/atık dönüşümü, bankacılar |
| **Kuzey Düğüm** | İleri-itici, kazanımcı | Rekabetçi sporcular, kapitalistler, iş insanları, liderler, arazi geliştiriciler |
| **Güney Düğüm** | İçe dönük, ruhsal, geçmişe ait | Astrologlar, natüropatlar, psikologlar, psişikler, keşişler, müzisyenler, sosyal çalışanlar, tarihçiler |

## 3. Üç vocational ev → çağrı alanı

| Ev | Hiyerarşi | Vocational anlam |
|---|---|---|
| **10. ev (MC)** | En yüksek | Çağrı, kariyer, itibar, statü, otorite, en büyük başarı |
| **6. ev** | İkinci | İşin kendisi: günlük iş, beceri, iş arkadaşları, iş ortamı |
| **2. ev** | Üçüncü | Kazanç gücü, gelir kaynağı, para |

> İstisna dışında 10 > 6 > 2 önceliği geçerlidir.

## 4. Burç → iş karakteri / çalışma tarzı (örnek örüntüler)

Eser her burcun "vocational rulership" ve "work faults" profilini verir. Quiz puanlaması için temel örüntüler:

| Burç | İş karakteri | Tipik çağrı eğilimi |
|---|---|---|
| Koç | Başlatıcı, cesur, hızlı | Öncülük, rekabet, fiziksel/keskin işler |
| Boğa | Sebatlı, konfor-sever, fiziksel dayanıklı | İstikrar isteyen, fazla mesai istemeyen işler |
| İkizler | İletişimci, çevik, çok yönlü | Yazı, iletişim, el becerisi, hareket |
| Yengeç | Besleyici, sadık, ev-merkezli | Bakım, aşçılık, herbalist, ebeveynlik, küçük işletme |
| Aslan | Karizmatik, sahne-seven, lider | Tiyatro, yöneticilik, dikkat çeken roller |
| Başak | Düzenli, detaycı, hizmet odaklı | Teknik, sekreterlik, analitik işler |
| Terazi | Estetik, ilişkisel, diplomatik | Tasarım, danışmanlık, sanat (bağımlılık riski notu var) |
| Akrep | Yoğun, gizlilik-seven, dönüştürücü | Araştırma, gizlilik gerektiren meslekler |
| Yay | Geniş, seyahat-sever, öğretici | Seyahat, öğretim, spor, hukuk |
| Oğlak | Protokol-sever, ciddi, yapısal | Yönetim, kurumsal, uzun-vadeli inşa |
| Kova | Yenilikçi, sabit-aile, özgün | Teknoloji, icat, havacılık, teori |
| Balık | Sezgisel, eksantrik, ruhsal | Sanat, ruhsal iş, antika; "sürekli çaba" zorluğu notu |

## 5. Element + Mode → çalışma davranışı

| Mode | İşlev | Çalışma davranışı |
|---|---|---|
| **Öncü (Cardinal)** — Koç, Yengeç, Terazi, Oğlak | Başlatma | Kendi başlatan, bağımsız, liderlik, organizasyon; takip eksik olabilir (toprak/Satürn destekler) → girişimci/serbest çalışan |
| **Sabit (Fixed)** — Boğa, Aslan, Akrep, Kova | Sürdürme | Takip-gücü, başkasının başlattığını bitirme, insan/para/kaynak yönetimi → bankacı, yönetici, işletme sahibi; esneklik/hız gereken işe uygun değil |
| **Değişken (Mutable)** — İkizler, Başak, Yay, Balık | Dağıtma | Genelde ne lider ne sürdürücü; en iyi yardımcı/uyarlanabilir rollerde, esneklik isteyen işlerde |

| Element | Çağrı alanı (Lilly'den) |
|---|---|
| Ateş | Ateşle çalışanlar — kuyumcu vb. |
| Toprak | Toprakla bağlantılı — çömlekçi, bahçıvan, inşaatçı |
| Hava | Şarkıcı, oyuncu, oyun bekçileri vb. |
| Su | Denizci, balıkçı, su işleri, hizmet |

## 6. Klasik yaklaşım ile modern uygulama bağlantısı (app için)

| Eserin yaklaşımı | Uygulamadaki karşılığı |
|---|---|
| Gezegen→meslek ailesi | Aylık arketip quizinde gezegen-arketip puanlaması |
| 10/6/2 ev önceliği | Kişiselleştirme algoritmasında "çağrı / iş / kazanç" katmanları |
| Element+mode çalışma tarzı | Kullanıcı iş-tarzı profili (öncü/sabit/değişken) |
| Üç-nokta okuma (Güneş+Ay+kariyer yöneticisi) | Çok-faktörlü arketip skoru için ağırlıklandırma |
