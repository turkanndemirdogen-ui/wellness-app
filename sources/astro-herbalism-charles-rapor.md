# Astro Herbalism (T.M. Charles) — Gezegen-Bitki Konsolidasyonu + Zamanlama Katmanı

> **Eser:** *Astro Herbalism: A Guide to Aligning the Stars and Plants for Health and Healing* — T.M. Charles
> **Modül:** 🌿 Herbalizm kasası. Modern, beden-pozitif, pratik. Ağır fence-off gerektirmez.
> **Değeri:** Kasadaki gezegen→bitki katmanını **doğrular ve konsolide eder** (Latince adlarla, modern gezegenler dahil) + app'in "gerçek-zamanlı gökyüzü" özelliğine doğrudan bağlanan **gezegensel zamanlama** katmanı ekler.
> **Standart uyarı:** Tıbbi tavsiye değil; toksikler işaretli/filtreli; zamanlama/ritüel geleneksel, kanıt değil.

## 1. Gezegen → bitki (konsolide, Latince adlarla)

Kasadaki diğer kaynaklarla (De Radiis, Raphael, Popham) **tutarlı**; modern gezegenleri de ekler.

| Gezegen | Tema | Bitkiler (toksik işaretli) |
|---|---|---|
| **Güneş** | Vitalite, kalp, bağışıklık | Sarı kantaron (*Hypericum perforatum*), biberiye (*Rosmarinus officinalis*), calendula (*Calendula officinalis*) |
| **Ay** | Duygu, sıvılar, üreme sistemi | Papatya (*Matricaria chamomilla*), melisa (*Melissa officinalis*), ak söğüt (*Salix alba*) |
| **Merkür** | İletişim, sinir sistemi, akciğer | Nane (*Mentha piperita*), lavanta (*Lavandula angustifolia*), rezene (*Foeniculum vulgare*) |
| **Venüs** | Sevgi, güzellik, böbrek, cilt | Menekşe (*Viola odorata*), gül, karahindiba |
| **Mars** | Enerji, eylem, inflamasyon, kas | Isırgan (*Urtica dioica*), zencefil (*Zingiber officinale*), kırmızı biber (*Capsicum annuum*) |
| **Jüpiter** | Genişleme, karaciğer, kan | Karahindiba (*Taraxacum officinale*), adaçayı (*Salvia officinalis*), meyan (*Glycyrrhiza glabra*) |
| **Satürn** | Yapı, kemik, cilt | Atkuyruğu (*Equisetum arvense*) · ⚠ comfrey/karakafes (*Symphytum*, hepatotoksik) · ⚠ siyah ceviz (*Juglans nigra*) |
| **Merkür (modern/sinir)** | Beyin, biliş | Ginkgo (*Ginkgo biloba*), gotu kola (*Centella asiatica*) · ⚠ kava (*Piper methysticum*, karaciğer riski) |
| **Neptün** | Rüya, sezgi, spiritüel | Valerian (*Valeriana officinalis*), mavi lotus (*Nymphaea caerulea*) · ⚠ mugwort (*Artemisia vulgaris*, gebelikte sakınılır) |
| **Plüton** | Dönüşüm, hormon, derin | Black cohosh (*Actaea racemosa*), mür (*Commiphora myrrha*) · ⛔ porsuk (*Taxus baccata*, **çok toksik — kesinlikle hariç**) |

## 2. ⭐ Gezegensel zamanlama katmanı (gerçek-zamanlı gökyüzü özelliği için)

Eserin app'e en özgün katkısı: içeriği **anlık gök verisine** bağlama mantığı.

| Zamanlama | İlke | App kullanımı |
|---|---|---|
| **Büyüyen Ay (waxing)** | Yapım/besleme/vitalite enerjisi | Güçlendirici tema: ısırgan, astragalus; "inşa et" içeriği |
| **Küçülen Ay (waning)** | Bırakma/arınma/detoks | Temizleyici tema: devedikeni (milk thistle), cleavers; "bırak" içeriği |
| **Gezegen günleri** | Pazar=Güneş, Pazartesi=Ay... | O günün gezegenine göre günlük içerik tonu |
| **Gezegen saatleri** | Her saat bir gezegen yönetir | İleri seviye; opsiyonel kişiselleştirme |

> Bu, senin Claude API + AstroAPI ile ürettiğin günlük dinamik içeriğin **çerçevesi**: natal veri (statik) + anlık ay fazı/gezegen günü (dinamik) → günün teması.

## 3. Element katmanı

| Element | Burçlar | Bitki niteliği |
|---|---|---|
| Ateş | Koç, Aslan, Yay | Isıtıcı, uyarıcı (kırmızı biber, zencefil) |
| Toprak | Boğa, Başak, Oğlak | Topraklayıcı, mineralize |
| Hava | İkizler, Terazi, Kova | Sinir-zihin, hafif aromatik |
| Su | Yengeç, Akrep, Balık | Soğutucu, nemlendirici, yatıştırıcı |

## 4. Duygusal/spiritüel + basit tarifler

- **Duygusal/spiritüel şifa:** gezegen/burç enerjisine göre yatıştırma, topraklama, sezgi desteği (mood + breathwork modülüne bağlanır).
- **Basit çay tarifleri:** nane, kekik, lavanta, melisa, papatya, alıç, rezene çayları — güvenli mutfak/çay bitkileri; tıbbi doz iddiası yok.

## 5. App entegrasyonu & sınırlar

| Modül | Bağlantı |
|---|---|
| Astro-semptom-herbal | Gezegen-bitki listesini doğrular/zenginleştirir (Latince adlı, filtreli) |
| Günlük içerik (gerçek-zaman) | Ay fazı + gezegen günü → günün teması |
| İçerik/eğitim | Basit çay tarifleri, element teması |
| Mood/breathwork | Duygusal/spiritüel şifa açısı |

| Sınırlama | Not |
|---|---|
| Toksikler | Porsuk **kesinlikle hariç**; comfrey, siyah ceviz, kava, mugwort uyarılı/filtreli |
| Zamanlama | Geleneksel; "kanıt" değil, içerik tonu çerçevesi |
| Klinik | Tıbbi tavsiye değil; tamamlayıcı/eğitsel |
