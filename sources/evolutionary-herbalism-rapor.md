# Evolutionary Herbalism (Sajah Popham) — Modern Gezegen-Bitki Enerjetik Çerçevesi

> **Eser:** *Evolutionary Herbalism* — Sajah Popham
> **Modül:** 🌿 Herbalizm kasası — **anahtar/omurga kaynak.** Astroloji↔tat↔enerjetik↔eylem köprüsünü modern, sistematik ve savunulabilir biçimde kurar.
> **Neden anahtar:** Kasadaki tarihsel kaynakları (Culpeper, Paracelsus, Raphael...) **modern klinik herbalizmle** birleştirir. Beden-pozitif, teşhissel değil; ağır fence-off gerektirmez. Standart uyarı: tıbbi tavsiye değil, toksikler hariç, tamamlayıcı düzey.

## 1. Çerçeve: Dört Doktrin + Beş Pillar

Popham, geleneksel enerjetik mimariyi dört doktrinle kurar:

| Doktrin | Anlam |
|---|---|
| **Emanasyon** | Tek kaynaktan kademeli açılım (kozmos→element→bitki) |
| **Correspondence** | "Yukarıda nasılsa, aşağıda öyle" — gök ve beden örüntülerinin karşılığı |
| **Signatura (imza)** | Bitkinin biçim/renk/tat/yeri iç doğasını gösterir (Paracelsus köprüsü) |
| **Sempatik tıp** | Benzer benzeri etkiler; tat ile gezegen/element arasındaki sempati ilişkisi |

## 2. Üç İlke (Tria Prima) ↔ Doshalar — Paracelsus köprüsü

| İlke | Karşılığı (dosha) | Doğa |
|---|---|---|
| **Tuz (Salt)** | Kapha | Beden/yapı, sabitleyici, nemli-toprak |
| **Kükürt (Sulfur)** | Pitta | Ruh/öz, dönüştürücü, ateş |
| **Cıva (Mercury)** | Vata | Zihin/hareket, uçucu, hava |

> Bu, önceki Paracelsus signatura raporundaki Tuz/Kükürt/Cıva ile **doğrudan örtüşür** — kasa içi tutarlılık.

## 3. Yedi Gezegen — arketipsel doğa (bitki + insan)

| Gezegen | Arketip | Bitki/insan teması |
|---|---|---|
| **Güneş** | Kral / merkezi vital ruh | Kalp, yaşam gücü, canlılık, merkezlenme (Kükürt özü) |
| **Ay** | Kraliçe / alıcı-yin | İç doğa, duygu, su/sıvı, ritim, besleme |
| **Mars** | Savaşçı / eylemdeki arzu | Kas, enerji, ısı, dürtü, hareket |
| **Venüs** | Sevgi / güzellik / ilişki | Uyum, çekim, duyusal algı, bağ |
| **Jüpiter** | Büyük Benefik / genişleme | Sevinç, vizyon, bolluk, yükseltme |
| **Satürn** | Sınır / kasılma / yapı | Maddeleştirme, sabitleme, kronik/yapısal, sınır |
| **Merkür** | Androjen / üçlü / haberci | Zihin, denge, iletişim, uçuculuk |

> Kutupluluklar: Güneş↔Ay, Mars↔Venüs, Jüpiter↔Satürn; Merkür denge noktası.

## 4. ⭐ Bir bitkiyi okumanın Beş Anahtarı (app'in omurgası)

Popham'ın pratik tanı-dışı çerçevesi — app'in astro-semptom-herbal akışının tam karşılığı:

| Anahtar | Ne yapar |
|---|---|
| **1. Tat (Taste)** | Bitki-beden ilk teması; enerjetik bilgisinin kapısı |
| **2. Affinite** | Bedende nereye etki eder (anatomik konum / "appropriations") |
| **3. Eylem (Action)** | Ne yapar (demulcent, stimülan, sedatif...) |
| **4. Enerjetik** | Altta yatan örüntü: ısıtıcı/soğutucu, nemlendirici/kurutucu |
| **5. Özel Potens** | Holistik/gestalt — bitkinin bütünsel imzası |

## 5. ⭐ Tat → İlke/Dosha → Element → Gezegen korelasyonu (kodlanabilir)

Bu tablo, app'in çekirdek eşleme motoru olabilir. Korelasyonlar **sempatik** (benzerlik) ilkesine dayanır.

| Tat | İlke / Dosha | Element | Gezegen(ler) | Eylem teması |
|---|---|---|---|---|
| **Tatlı** | Tuz / Kapha | Toprak, Su | Ay, Venüs, Jüpiter | Nemlendirici; besleyici, demulcent, emollient; mukoza affinitesi |
| **Ekşi** | Kükürt / Pitta | Toprak, Ateş | Güneş, Venüs | Isıtıcı, asidik |
| **Tuzlu** | Tuz / Kapha | Su, Ateş | Venüs, Satürn, Ay | Mineral, ağır |
| **Keskin/Acımsı (pungent)** | Kükürt / Pitta | Ateş, Hava | Güneş, Mars | Isıtıcı; durağanlık/soğuğu dağıtır, sindirim+dolaşım, antiseptik |
| **Acı (bitter)** | Cıva / Vata | Hava, Eter | Merkür, Satürn, Jüpiter | Çoğunlukla soğutucu; alteratif/sindirim/nervin alt tipleri; sedatif |
| **Buruk (astringent)** | Cıva / Vata | Toprak, Hava | Satürn | Tanin; sıkılaştırıcı/kurutucu; gevşek dokuyu toparlar |

> **Tasarım fırsatı:** Kullanıcının semptom/enerjetik durumu → eksik kutbu dengeleyen **tat** → o tada karşılık gelen güvenli bitki. Bu, app'in kişiselleştirme mantığını tat-enerjetik üzerinden somutlaştırır.

## 6. App entegrasyonu

| Modül | Bağlantı |
|---|---|
| Astro-semptom-herbal | Beş Anahtar + tat-korelasyon tablosu = öneri motorunun çerçevesi |
| Kişiselleştirme | Gezegen arketipi → enerjetik ihtiyaç → dengeleyici tat/bitki |
| İçerik/eğitim | "Bir bitkiyi nasıl okursun" (tat→enerjetik) eğitici içerik |
| Mood / vitalite | Yedi gezegen arketipi → nazik öz-bakım tonu |

## 7. Sınırlamalar

| Konu | Not |
|---|---|
| Klinik | Tıbbi tavsiye değil; tamamlayıcı/eğitsel düzey |
| Toksik | Siyanojenik acılar (Yaban kirazı, şeftali yaprağı vb.) dikkat; toksikler öneri motorunda filtreli |
| Enerjetik model | Geleneksel-modern köprü; fitokimya ile birebir değil ama klinik herbalist çerçevesi sağlam |
| Sempatik korelasyon | Tat↔gezegen ilişkisi sembolik-sempatik; "kanıt" değil çerçeve |
