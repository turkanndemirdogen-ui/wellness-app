# Classical Medical Astrology (Oscar Hofman) — Mizaç & Önleme: Denge Omurgası

> **Eser:** *Classical Medical Astrology* — Oscar Hofman
> **Modül:** ✨ Kişiselleştirme + 🌿 herbalizm + 💭 mood + beslenme + skincare/renk. Modern, klasik-temelli, titiz.
> **Neden omurga:** Dört-humor/mizaç sistemini ve özellikle **önleme (prevention)** yaklaşımını sunar — teşhis değil, *"mizacın şu yöne yatıyor, şununla dengele."* Kasadaki tüm "denge" mantığını **tek konstitüsyonel çerçevede** birleştirir. Beden-pozitif, yaşam-tarzı temelli.
> **⚠️ Sınır:** Decumbiture/medikal horary teşhis, ameliyat soruları, medikal elections (tıbbi zamanlama), hastalık adlandırma → app **dışı.**

## 1. Dört humor / element / nitelik (konstitüsyon çerçevesi)

İki eksen: **sıcak↔soğuk** (birincil), **nemli↔kuru** (ikincil).

| Humor | Element | Nitelik | Mizaç | Mevsim/yaşam |
|---|---|---|---|---|
| Sarı safra (yellow choler) | Ateş | Sıcak + kuru | Koleric: aktif, atılgan, çabuk | Yaz; ergenlik 2. faz |
| Kara safra (black choler) | Toprak | Soğuk + kuru | Melankolik: içe dönük, ağır | Sonbahar |
| Balgam (phlegm) | Su | Soğuk + nemli | Flegmatik: yumuşak, duygusal, çatışmadan kaçınır | Kış; gebelik (flegmatik faz) |
| Kan (blood) | Hava | Sıcak + nemli | Sanguin: neşeli, enerjik, sosyal | İlkbahar |

## 2. ⭐ Mizaç → Önleme → Denge (app'in anahtar çerçevesi)

Hofman'ın önleme mantığı tam app'e göre: **natal mizaç okunur, dengeleyici yaşam-tarzı önerilir.** Teşhis değil.

> İlke: **"Humorlar ne kadar dengeliyse konstitüsyon o kadar güçlü."** Bir humor fazlaysa (ör. sarı safra), karşıt nitelik eklenir.

| Mizaç eğilimi | Fazlalık riski | Dengeleyici öz-bakım (örnek) |
|---|---|---|
| Koleric (sıcak-kuru) | Ateşli/aşırı-aktif eğilim | Nemlendirici-soğutucu gıdalar, bol hareket, mavi/yeşil taş/renk |
| Melankolik (soğuk-kuru) | Ağırlık, içe kapanma | Sıcak-nemli (sanguin) gıdalar, ışık, bağlantı |
| Flegmatik (soğuk-nemli) | Durağanlık, fazla pasiflik | Isıtıcı-kurutucu, hareket, uyarım |
| Sanguin (sıcak-nemli) | Dağılma, aşırılık | Topraklama, ölçü, dinlenme |

> **Tasarım:** Bu, kasadaki "eksik karşıt enerjiyi ekle" ilkesinin **konstitüsyonel formu.** App, natal mizacı (element/gezegen dağılımı) okur → eksik kutbu gıda/hareket/renk ile nazikçe dengeler.

## 3. Mevsimsel telafi (gerçek-zamanlı içerik)

Humorlar mevsimle bağlı; app mevsime göre **telafi** önerebilir:
- Sonbahar (melankolik) → sıcak-nemli sanguin gıdalar, "sonbahar dip"ini yumuşatma.
- Yaz (koleric) → soğutucu-nemli (ör. soğuk çorbalar, salatalık/domates).
- Kış (flegmatik) → ısıtıcı-uyarıcı.
- İlkbahar (sanguin) → hafif, dengeleyici.

> Mevsim + mizaç → günlük içerik tonu. AstroAPI'den mevsim/güneş konumu + natal mizaç birleşir.

## 4. ⚠️ Wellbeing çerçevesi (kritik)

Hofman "flegmatik depresyon" / "melankolik" terimlerini kullanır. App bunları **asla tanı olarak** kullanmaz:
- Melankolik/flegmatik eğilim = **doğal mizaç**, depresyon etiketi değil.
- "Sonbahar dip" = mevsimsel **nazik öz-bakım** vesilesi (ışık, sıcak gıda, bağlantı), klinik durum değil.
- Hiçbir kullanıcıya "depresif mizaçlısın" denmez; yalnız "şu denge sana iyi gelebilir."

## 6. App entegrasyonu

| Modül | Bağlantı |
|---|---|
| **Kişiselleştirme** | Natal mizaç → dengeleyici içerik (konstitüsyonel denge) |
| **Beslenme** | Sıcak/soğuk-nemli/kuru gıda dengesi (beden-nötr, kilo-dışı) |
| **Hareket** | Mizaca göre hareket önerisi (koleric→serinleten, flegmatik→uyaran) |
| **Skincare/renk** | Dengeleyici renk/taş (mavi-yeşil serinletir vb.) |
| **Mood** | Mizaç → nazik öz-bakım tonu (tanı değil) |
| **Gerçek-zaman** | Mevsim + mizaç → günün teması |

## 7. Sınırlamalar

| Konu | Not |
|---|---|
| Wellbeing | Melankolik/flegmatik = mizaç, depresyon değil; tanı yok |
| Klinik | Önleme/yaşam-tarzı düzeyi; tıbbi tavsiye değil |
| Decumbiture/ameliyat/elections | Fence-off |
| Renk/taş | Sembolik/geleneksel; tıbbi iddia yok |
